StorefrontPayment

StorefrontPayment(parent, collectionopt, paymentTypeopt, storefrontTypeopt) → {Promise.<StorefrontPayment>}

The StorefrontPayment class is the base class for all storefront payment methods, which implements generic and reusable functions. Override the functions in this class if customizations for the payment is needed.
